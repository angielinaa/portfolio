# 🐍 Python Mini-Projects & Explorations

Welcome to my Python portfolio! This repository contains a collection of mini-projects I've built to strengthen my programming logic, explore graphical user interfaces (GUI), and dive into Computer Vision.

## 📂 Featured Projects

### 1. 🖐️ Gesture Vocalizer
An experimental Computer Vision project utilizing Python. This script tracks hand gestures using a webcam and translates specific movements into actions/sounds.
- **Tech Stack:** Python, OpenCV, MediaPipe

### 2. 🧮 GUI Calculator
A fully functional calculator application with a clean graphical user interface. Built to understand event-driven programming and state management in desktop apps.
- **Tech Stack:** Python, Tkinter (or PyQt)

### 3. 🐍 Classic Snake Game
A recreation of the classic arcade game. This project helped me solidify my understanding of game loops, collision detection, and object-oriented programming (OOP).
- **Tech Stack:** Python, Pygame (or Turtle)

## 🚀 How to Run
1. Clone this repository: `git clone https://github.com/angielinaa/portfolio.git`
2. Navigate to the specific project folder.
3. Install the required dependencies (if any).
4. Run the main python script: `python filename.py`
